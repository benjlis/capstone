Global Legal Entity Identifier (LEI) Search Utility

LEI Query: A Thinkful Capstone Project

Developer: Ben Lis, Mentor: Brian Redfern

What it does:

LEI Query enables registered users to query the 400K+ record Legal Entity Identifier database for companies by either their name or LEI code.

If you've got an account, you can give it a try by entering the name of your favorite large tech company such as Google into the name field. You can see an even larger result set by typing in the name of a large investment firm such as Goldman Sachs.

Built with this technology stack:

Python
general purpose programming language, v2.7
Flask
web framework that includes Jinja 2 and SQLAlchemy, v.10
PostgreSQL
relational database management system, v9.3
Bootstrap
HTML5, CSS, JavaScript framework for developing responsive front ends, v3.3

Behind the scenes:

  • Copies and unzips the 50Mb GLEIF XML file
  • XML parses and loads the 400K LEI records from the GLEIF file into Postgresql
  • SQLAlchemy data models for users and legal entities
  • User management: creates new registerd users, login and logout functionality
  • Support for query by LEI or Name
  • Jinja2 templates with Bootstrap support for the landing and query pages
  • Widget of LEI Smart twitter feed

To learn more about LEIs check out:

  • Global Legal Entity Identifier Foundation (GLEIF)
  • Legal Entity Identifier Regulatory Oversight Committee (LEI ROC)
